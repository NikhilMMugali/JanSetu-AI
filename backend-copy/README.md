# Backend Server

This directory contains the core backend server files for JanSetu AI.

## Files

- **server.js** - Express.js server with Vision API integration
- **index.ts** - TypeScript entry point
- **package.json** - Dependencies configuration
- **tsconfig.json** - TypeScript compiler options

## About This Structure

This is part of the hackathon milestone reorganization. Original files remain in root for Vercel deployment compatibility.

- Original server files: `/server.js`, `/package.json`, `/tsconfig.json`
- Backend copy: `/backend-copy/` (this directory)
- Deployment active: `/` (root level)

See root README.md for full project structure and setup instructions.
